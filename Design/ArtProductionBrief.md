# 소울 아처 아트 제작 브리프

## 문서 상태

- 상태: `draft`
- 제작 담당: 회사 이미지 Worker
- 검토: Designer 정합성 검토 후 사용자 승인
- 구현 인계: 승인된 최종 PNG와 스프라이트 시트를 Unity 구현 저장소에 전달

## 목표

정통 영웅 판타지 분위기의 일관된 픽셀아트를 제작한다. 모든 플레이 자산은 640×360 내부 해상도와 1920×1080 정수 3배 출력에서 선명하게 보여야 한다.

## 공통 규격

- 타일: 16×16픽셀
- 플레이어와 일반 몬스터: 32×32픽셀
- 보스: 64×64픽셀
- 투명 배경 PNG
- nearest-neighbor 확대에 적합한 단단한 픽셀 경계
- 안티앨리어싱, 반투명 외곽선과 자동 리사이즈 흔적 금지
- 동일 인물의 팔레트, 비율과 장비 위치를 모든 프레임에서 유지
- 공격 예고는 색상뿐 아니라 형태와 점멸을 함께 사용

## 공통 팔레트

- 게임 전체에 최대 32색의 공통 마스터 팔레트를 사용한다.
- 숲, 동굴과 마왕의 성은 마스터 팔레트 중 12~16색을 중심으로 사용한다.
- 레온, 엘리아와 모르칸의 대표 색은 지역이 바뀌어도 같은 팔레트 값을 유지한다.
- 적 공격 경고는 붉은 계열, 자동 조준 고리는 금색, 왕가의 빛은 흰색·금색·파란색, 영혼 조각은 밝은 청록 계열로 기능 색을 고정한다.
- 기능 색은 배경 장식의 넓은 면에 사용하지 않아 HUD와 전투 효과의 가독성을 유지한다.
- 색상만으로 상태를 구분하지 않고 아이콘, 윤곽과 애니메이션을 함께 사용한다.
- 정확한 RGB·HEX 값은 첫 캐릭터 스타일 시트 검토에서 확정한다.

## 제작 대상

### 캐릭터

- 레온: 파란 망토, 갈색 가죽 장비, 밝은 머리, 나무 활
- 엘리아: 흰 드레스, 금색 장식, 파란 보석, 작은 왕관
- 모르칸: 검은 갑옷, 붉은 망토·눈, 뿔 투구, 대검과 방패

### 일반 몬스터

- 숲 늑대, 독버섯
- 바위 딱정벌레, 수정 정령
- 방패 해골병, 암흑 마법사
- 각 몬스터는 정면 기준 이동 2프레임과 피격 1프레임을 제작한다.
- 별도 공격 프레임은 만들지 않고 `띠딕` 점멸, 예고선과 투사체 효과를 사용한다.
- 별도 사망 프레임 대신 모든 일반 몬스터가 공유하는 영혼 해방 효과를 사용한다.

### 보스

- 거대 나무 정령
- 거대 박쥐
- 마왕 기사 모르칸 1단계와 갑옷이 파괴된 2단계

#### 보스 프레임 기준

- 모든 보스는 대기 2프레임과 공통 피격 점멸을 사용한다.
- 패턴마다 준비 자세 1프레임을 만들고 실제 공격 움직임은 투사체, 예고선과 효과 애니메이션으로 표현한다.
- 거대 나무 정령: 뿌리 준비, 씨앗 준비, 식물 소환 자세를 각 1프레임 제작한다.
- 거대 박쥐: 비행 대기 2프레임, 돌진 준비, 음파 준비, 암전 자세, 벽 충돌 기절을 각 1프레임 제작한다.
- 모르칸: 방패 대기 2프레임, 검기 준비, 방패 해제, 갑옷 파괴 전환 2프레임, 2단계 대기 2프레임, 연속 참격·교차 검기 준비를 각 1프레임 제작한다.
- 보스별 개별 사망 애니메이션 대신 큰 영혼 해방 효과와 화면 흔들림을 공통 사용한다.
- 실제 스프라이트 시트는 동일 보스 안에서 프레임 크기와 기준점을 고정한다.

### 환경

- 숲 타일 세트와 전투방 3개
- 동굴 타일 세트와 전투방 3개
- 마왕의 성 타일 세트와 전투방 3개
- 입장방, 상점방, 보스방, 영혼 상자, 치유의 석상, 영혼의 샘

### UI와 효과

- 하트, 영혼 조각, 미니맵, 조준 고리, 능력 6종, 상점 상품 3종 아이콘
- 화살, 벽 반사, 피격, 영혼 획득, 문 개방과 왕가의 빛
- 일반 공격 점멸과 `띠딕` 경고에 대응하는 시각 효과

### 일러스트

- 도입 컷신 3장: 만남, 납치, 출발
- 엔딩 컷신 2장: 구출, 귀환
- 타이틀 배경 1장: 레온이 멀리 있는 마왕의 성을 바라보는 장면

## 제작 순서

1. 레온·엘리아·모르칸 캐릭터 스타일 시트와 공통 팔레트
2. 레온 8방향 걷기 기준과 기본 화살
3. 일반 몬스터 6종
4. 보스 3종과 패턴 효과
5. 지역별 타일 세트와 방 소품
6. HUD·능력·상점 아이콘
7. 타이틀과 컷신 일러스트

## 검수 기준

- 원본 크기에서 캐릭터와 공격 방향을 구분할 수 있다.
- 레온은 세 지역 배경에서 항상 식별된다.
- 일반 몬스터 6종의 실루엣이 서로 겹치지 않는다.
- 보스 패턴의 안전 영역과 위험 영역이 정지 화면에서도 구분된다.
- 스프라이트 시트 프레임 크기와 기준점이 일정하다.
- Unity 임포트 후 픽셀이 흐려지거나 프레임이 흔들리지 않는다.

## 인계 전 미결정

- 32색 공통 팔레트의 정확한 RGB·HEX 값
- 컷신 일러스트의 내부 제작 해상도
