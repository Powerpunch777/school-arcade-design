# 소울 아처 제작 범위

## 문서 상태

- 상태: `approved`
- 승인일: 2026-07-13
- 목적: 일정이 부족할 때 게임의 핵심을 유지하면서 범위를 줄이는 순서를 정의한다.

## 핵심 원칙

3개 스테이지, 지역별 보스와 이동·정지 자동 공격은 소울 아처의 정체성이므로 유지한다. 일정 문제는 부가 방, 통계 표시와 추가 연출을 줄여 해결한다.

## 필수 범위

- Windows PC 실행 파일
- 한국어 UI
- 이동 중 회피, 정지 시 화면 안의 가장 가까운 적 자동 공격
- 하트 5개, 피격, 사망과 처음부터 재시작
- 숲, 동굴, 마왕의 성 3개 스테이지
- 스테이지당 전투방 3개, 상점방 1개, 보스방 1개
- 일반 몬스터 6종과 보스 3종
- 플레이어 능력 6종, 한 판에 6회 선택, 최대 3중첩
- 영혼 조각과 간단한 상점
- 자유 탐색, 방 상태 유지와 미니맵
- 도입 컷신 3장과 공주 구출 엔딩 1장
- 쉬움과 보통 난이도
- 판 종료 결과 기록

## 목표 범위

- 50% 확률로 생성되는 보상·회복·이벤트방
- 최근 20판 전적 목록
- 승률, 평균 시간, 평균 도달 스테이지와 선호 능력 요약 통계
- 왕국 귀환 엔딩 일러스트 1장
- 전체 음량과 전체화면 설정
- 스테이지별 배경음악 3곡

## 축소 순서

일정 위험이 확인되면 아래 순서대로 제외한다. 앞 단계만으로 충분하면 다음 단계는 유지한다.

1. 이벤트방을 제거하고 추가 방을 보상방 또는 회복방으로 대체한다.
2. 전적의 요약 통계를 제거하고 개별 판 기록 목록만 유지한다.
3. 왕국 귀환 엔딩 장면을 제거하고 공주 구출 1장만 유지한다.
4. 선택 특수방 전체를 제거해 스테이지를 고정 5개 방으로 만든다.
5. 설정 메뉴를 제거하고 기본 창 모드와 고정 음량을 사용한다.

## 축소 금지

- 스테이지나 보스를 제거하지 않는다.
- 이동·정지 자동 공격 규칙을 바꾸지 않는다.
- 일반 몬스터를 지역 간 단순 색상 변경으로 대체하지 않는다.
- 마왕 기사 처치와 공주 구출 결말을 제거하지 않는다.

## 일정 판단 시점

- 숲 스테이지 수직 단면 완성 후 전체 예상 작업량을 다시 계산한다.
- 동굴 스테이지 완료 시 목표 범위의 남은 작업을 평가한다.
- 최종 제출 1주 전에는 새 기능을 추가하지 않고 버그 수정과 밸런스에 집중한다.
