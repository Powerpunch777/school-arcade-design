# 화면 및 상태 흐름

## 문서 상태

`draft`

## 전체 흐름

```text
타이틀
  -> 조작법
  -> 타이틀
  -> 새 게임
  -> 난이도 선택: 쉬움 / 보통
  -> 도입 컷신 3장
  -> 시작방
  -> 스테이지 1: 숲 자유 탐색
       -> 전투
       -> 클리어
       -> 능력/보상 선택
       -> 다음 방 또는 이전 방
  -> 숲 보스
  -> 스테이지 출구
  -> 스테이지 2: 동굴 자유 탐색
  -> 동굴 보스
  -> 스테이지 출구
  -> 스테이지 3: 마왕의 성 자유 탐색
  -> 마왕 기사
  -> 공주 구출
  -> 엔딩 컷신 2장
  -> 승리 결과

전투 중 체력 0
  -> 패배 결과
  -> 다시 시작 / 타이틀
```

## 게임 상태

| 상태 | 플레이어 입력 | 시간 진행 | 전환 조건 |
|---|---|---|---|
| 타이틀 | 새 게임 선택 | 정지 | 새 게임 |
| 조작법 | 뒤로 | 정지 | 타이틀 |
| 난이도 선택 | 쉬움/보통 선택, 뒤로 | 정지 | 도입 컷신 또는 타이틀 |
| 도입 컷신 | 다음, 건너뛰기 | 정지 | 3장 완료 또는 건너뛰기 |
| 엔딩 컷신 | 다음 | 정지 | 2장 완료 |
| 방 진입 | 없음 | 정지 | 신규 전투 시작 또는 클리어 상태 복원 |
| 전투 | 이동, 일시정지 | 진행 | 적 전멸 또는 사망 |
| 방 클리어 | 없음 | 진행 | 문 개방 완료 |
| 능력 선택 | 카드 선택 | 정지 | 1개 선택 |
| 특수방 | 선택 또는 구매 | 필요에 따라 정지 | 보상 확정 |
| 일시정지 | 메뉴 선택 | 정지 | 계속/재시작/나가기 |
| 결과 | 다시 시작/타이틀 | 정지 | 버튼 선택 |

## 전투 HUD 배치

```text
+--------------------------------------------------+
| HP [♥ ♥ ♥ ♥ ♡]             ROOM 04/18   SOUL 12 |
|                                                  |
|                 게임 플레이 영역                |
|                                                  |
| [능력 아이콘]             [미니맵]    [PAUSE]    |
+--------------------------------------------------+
```

HUD는 플레이 영역을 가리지 않으며 고정된 크기를 유지한다. 능력 설명은 아이콘에 마우스를 올렸을 때 표시한다.
