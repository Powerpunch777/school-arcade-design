# 소울 아처 UI 와이어프레임

## 문서 상태

- 상태: `draft`
- 내부 기준 해상도: 640×360
- 기본 출력: 1920×1080 FHD, 정수 3배 확대

## 공통 원칙

- UI 텍스트는 한국어만 사용한다.
- 픽셀 그래픽과 한글이 흐려지지 않도록 정수 배율을 유지한다.
- 버튼은 키보드 포커스와 마우스 호버 상태를 함께 제공한다.
- 선택 상태는 색상뿐 아니라 테두리와 작은 화살표로 표시한다.
- 화면마다 뒤로 가기와 확정 동작을 같은 위치와 입력 규칙으로 유지한다.

## 타이틀 화면

레온이 숲의 높은 지대에서 멀리 있는 마왕의 성을 바라보는 전체 화면 픽셀 배경을 사용한다. 성은 우측 상단, 레온은 우측 하단에 배치하고 좌측은 로고와 메뉴를 읽을 수 있는 비교적 단순한 하늘과 숲 배경으로 남긴다.

```text
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│   SOUL ARCHER                              [마왕의 성]       │
│   소울 아처                                                   │
│                                                              │
│   > 새 게임                                                   │
│     전적                                                     │
│     조작법                                                   │
│     설정                                                     │
│     종료                                      [레온]         │
│                                                              │
│   버전 표기                                                   │
└──────────────────────────────────────────────────────────────┘
```

### 구성

- 영문 `SOUL ARCHER`와 한글 `소울 아처`를 좌측 상단에 함께 표시한다.
- 메뉴는 좌측 중앙에 세로로 배치한다.
- 메뉴 순서는 `새 게임`, `전적`, `조작법`, `설정`, `종료`다.
- 기본 포커스는 `새 게임`이며 위·아래 방향키와 마우스로 이동할 수 있다.
- 선택 항목은 금색 테두리, 좌측 화살표와 밝아진 글자로 표시한다.
- `Enter`, `Space` 또는 클릭으로 선택하고 `Esc`는 하위 화면에서 타이틀로 돌아올 때만 사용한다.
- `종료` 선택 시 확인 창을 표시한다.
- 버전 표기는 좌측 하단에 작게 두고 게임 제목이나 메뉴보다 눈에 띄지 않게 한다.

### 배경 연출

- 레온의 망토와 나뭇잎에 2~3프레임의 느린 바람 애니메이션을 반복한다.
- 마왕의 성 창문에는 약한 붉은 빛을 사용하지만 메뉴 글자보다 밝지 않게 한다.
- 카메라 이동, 패럴랙스와 복잡한 입자 효과는 사용하지 않는다.
- 배경 음악은 숲 테마의 잔잔한 도입부를 반복한다.

## 플레이 HUD

```text
┌──────────────────────────────────────────────────────────────┐
│ ♥ ♥ ♥ ♥ ♡     숲  2/3                         ┌─┬─┬─┐      │
│ ◇ 영혼 12                                     │□│■│?│      │
│                                                ├─┼─┼─┤      │
│                                                │ │◆│ │      │
│                                                └─┴─┴─┘      │
│                                                              │
│                    게임 플레이 영역                         │
│                                                              │
│                                                              │
│ [능력 아이콘 1][능력 아이콘 2][능력 아이콘 3]       [Ⅱ]    │
└──────────────────────────────────────────────────────────────┘
```

### HUD 구성

- 하트 5개는 좌측 상단 첫 줄에 표시한다.
- 현재 스테이지 이름과 클리어한 전투방 수를 하트 오른쪽에 표시한다.
- 영혼 조각 수는 좌측 상단 둘째 줄에 영혼 아이콘과 함께 표시한다.
- 보유 능력 아이콘은 좌측 하단에 획득 순서대로 배치하고 중첩 수를 작은 숫자로 표시한다.
- 일시정지 버튼은 우측 하단에 아이콘으로 표시하며 `Esc`와 같은 기능을 수행한다.
- HUD 요소는 화면 가장자리에서 일정한 안쪽 여백을 유지한다.

### 상시 미니맵

- 미니맵은 우측 상단에 항상 표시하며 별도 열기 버튼은 사용하지 않는다.
- 현재 방은 금색 마름모, 방문한 방은 채워진 칸, 발견한 미방문 방은 윤곽 칸으로 표시한다.
- 상점, 보상과 회복방은 발견한 뒤에만 각기 다른 아이콘으로 표시한다.
- 발견하지 않은 보스방은 일반 미방문 방과 같게 보인다.
- 미니맵의 배경은 반투명한 어두운 면을 사용하되 전투 예고를 완전히 가리지 않는다.
- 화면 밖 공격 경고가 우측 상단에서 발생하면 미니맵 바깥쪽에 표시해 겹치지 않게 한다.

## 능력 선택

전투 화면을 어둡게 덮고 같은 크기의 카드 3개를 중앙에 표시한다. 선택 중에는 게임 시간이 정지한다.

```text
┌──────────────────────────────────────────────────────────────┐
│                    능력 하나를 선택하세요                    │
│                                                              │
│      [1]                 [2]                 [3]             │
│  ┌────────────┐      ┌────────────┐      ┌────────────┐     │
│  │   [활↑]    │      │   [시계]   │      │ [관통 화살]│     │
│  │ 공격력 증가│      │공격 속도 증가│     │    관통    │     │
│  │    1/3     │      │    0/3     │      │    2/3     │     │
│  │ 공격력 +20%│      │간격 -15%   │      │관통 +1     │     │
│  └────────────┘      └────────────┘      └────────────┘     │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

- 마우스 클릭 또는 숫자키 `1`, `2`, `3`으로 선택한다.
- 첫 입력 직후 모든 카드를 비활성화해 중복 선택을 막는다.
- 선택한 카드는 짧게 금색으로 강조하고 나머지 카드를 숨긴 뒤 플레이로 돌아간다.
- 카드 크기와 텍스트 줄 수는 세 능력 모두 동일하게 유지한다.
