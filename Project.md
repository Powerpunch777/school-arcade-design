# School Arcade Project

## 상태

`draft` - 사용자 검토 전이며 구현 승인 상태가 아닙니다.

## 한 줄 설명

이동 중에는 회피하고, 멈추면 가장 가까운 적을 자동 공격하는 탑다운 픽셀 로그라이트 액션 게임.

## 목표

- 짧은 한 판 안에서 이동, 자동 공격, 능력 선택의 재미를 전달한다.
- 방 종류와 성장 선택을 확률적으로 구성해 반복 플레이에 변화를 준다.
- 플레이 결과를 수집하여 확률과 난이도를 통계적으로 개선한다.
- Unity와 C#을 사용해 실제 실행 가능한 2D 프로토타입을 완성한다.

## 대상 플랫폼

- 1차: Windows PC, 키보드 조작
- 2차 후보: Unity WebGL
