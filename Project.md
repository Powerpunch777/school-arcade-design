# 소울 아처 (Soul Archer)

## 상태

`draft` - 사용자 검토 전이며 구현 승인 상태가 아닙니다.

## 한 줄 설명

우연히 만난 공주를 구하기 위해 초보 궁수가 모험을 떠나는 탑다운 픽셀 로그라이트 액션 게임. 이동 중에는 회피하고, 멈추면 가장 가까운 적에게 화살을 자동 발사한다.

## 제목

- 한국어·영문 표기: `소울 아처 (Soul Archer)`
- `소울`은 몬스터를 처치해 얻고 상점에서 사용하는 영혼 조각을 의미한다.
- 배포 전 동명 게임과 상표 혼동 가능성을 확인한다.

## 목표

- 짧은 한 판 안에서 이동, 자동 공격, 능력 선택의 재미를 전달한다.
- 방 종류와 성장 선택을 확률적으로 구성해 반복 플레이에 변화를 준다.
- 숲, 동굴, 마왕의 성으로 이어지는 3개 스테이지를 완주한다.
- 플레이 결과를 수집하여 확률과 난이도를 통계적으로 개선한다.
- Unity와 C#을 사용해 실제 실행 가능한 2D 프로토타입을 완성한다.

## 대상 플랫폼

- Windows PC 실행 파일, 키보드 조작
- WebGL과 모바일은 현재 프로젝트 범위에서 제외
