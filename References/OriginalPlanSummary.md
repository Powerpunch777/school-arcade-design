# 원 계획서 요약

## 출처

`3607 손영하-융프 계획서.hwpx`, 학번 3607, 손영하

## 주요 내용

- 참고 게임: 궁수의 전설, Soul Knight
- 자동 공격과 방 단위 로그라이트 진행 결합
- 전투방 60%, 보상방 15%, 회복방 10%, 상점방 10%, 이벤트방 5%
- 마지막 방은 보스방으로 고정
- 방 클리어 후 공격력, 공격 속도, 관통, 반사, 회복, 이동 속도 중 무작위 3개 제시
- 픽셀아트 캐릭터, 적, 아이템, 타일, UI 직접 제작
- 플레이테스트로 클리어 시간, 사망 방, 선택 능력, 체력과 만족도를 기록
- 확률과 통계 및 미술을 게임 제작과 융합

## 정정 필요 항목

- 표지 주제 `룰 변형 보드게임 제작`은 본문의 액션 웹게임 내용과 불일치한다.
- HTML, CSS, JavaScript 기술 계획은 Unity와 C# 기준으로 수정해야 한다.
